# HandWriting
MFC框架实现简单手写字符识别

## 项目简介
### 介绍
项目旨在基于机器学习算法实现手写字符（包括数字和大小写字母）的识别。该系统通过读入一张字符图片的方式，实现识别对象的输入，通过BP神经网络、支持向量机等实现手写字符的识别，并输出识别的结果。
### 内容
* 1）手写字符的输入
* 2）图片预处理与特征提取
* 3）输出识别结果和图像信息

### 开发环境
* A）Visual Studio 2013
* B）MNIST 数据集
